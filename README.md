# Data Analysis & Quantitative Finance Projects

This repository contains projects related to data analysis, statistics, machine learning, and quantitative finance using Python.

## About

I hold an MSc in Pure Mathematics and have a strong background in probability, statistics, and mathematical modeling. Currently, I am transitioning into Data Analytics and Risk Analytics, applying quantitative methods to real-world financial and data-driven problems.

## Projects

### Apple Stock Analysis

Exploratory analysis of Apple stock using historical market data.

Topics covered:

* Financial data collection with yfinance
* Log returns
* Volatility analysis
* Return distributions
* Statistical analysis and visualization

### Quantitative Portfolio Optimization & Risk Analysis

A portfolio optimization project based on Modern Portfolio Theory.

Topics covered:

* Log returns and volatility
* Correlation and covariance matrices
* Portfolio return and risk
* Sharpe Ratio
* Monte Carlo portfolio simulation
* Efficient Frontier
* Beta estimation
* CAPM
* Security Market Line (SML)
