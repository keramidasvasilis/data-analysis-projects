# Quantitative Portfolio Optimization & Risk Analysis

## Overview

This project applies quantitative finance techniques to construct and analyze a diversified stock portfolio using historical market data.

The analysis combines statistical methods, portfolio theory, and financial modeling in order to study the relationship between risk and return.

## Assets

The portfolio consists of the following stocks:

* Apple (AAPL)
* Microsoft (MSFT)
* Johnson & Johnson (JNJ)
* JPMorgan Chase (JPM)
* Coca-Cola (KO)
* Exxon Mobil (XOM)

The S&P500 index is used as a market benchmark.

## Topics Covered

* Historical price analysis
* Log returns
* Volatility analysis
* Correlation and covariance matrices
* Portfolio return and risk
* Sharpe Ratio
* Monte Carlo portfolio simulation
* Efficient Frontier
* Beta estimation through linear regression
* Capital Asset Pricing Model (CAPM)
* Security Market Line (SML)

## Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* SciPy
* yfinance

## Key Findings

* Diversification significantly reduces portfolio risk.
* Efficient portfolios can be identified through Monte Carlo simulation.
* CAPM provides a theoretical framework linking expected return to systematic risk.
* Historical returns may deviate substantially from CAPM predictions due to factors beyond systematic market exposure.
