# Apple Stock Analysis

## Overview

This project explores the historical performance of Apple Inc. (AAPL) stock using Python and publicly available market data.

The objective is to analyze price behavior, returns, volatility, and statistical properties of stock returns through data visualization and exploratory data analysis.

## Topics Covered

* Historical stock price analysis
* Daily returns
* Log returns
* Volatility analysis
* Return distributions
* Skewness and kurtosis
* Data visualization

## Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* yfinance

## Key Findings

* Log returns provide a convenient framework for financial analysis.
* Stock returns exhibit significant variability over time.
* Return distributions may deviate from normality and display asymmetry and heavy tails.
